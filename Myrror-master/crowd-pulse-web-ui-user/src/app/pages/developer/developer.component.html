Access Token

Use access-token to use {{appName}} API.

You can use your access token in three ways:

  1. In the body of a POST request with access_token field
  2. With x-access-token header in the HTTP request
  3. In the URL with access_token query parameter

Endpoints

Profile

/profile/username

GET request to retrieve the profile information of the specified user. Only the information shared by the users can be retrieved.
